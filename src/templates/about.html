What is keybase?

This software is an Open Source Knowledge Base offering smart searches and recommendations based on Redis Stack. Using this software:

  • You can browse, search, and bookmark Knowledge Base articles
  • Consult recommended similar documents
  • Create and edit documents using the WYSIWYG editor
  • Markdown is also supported by the editor
  • Documents can be tagged

Roles

  • Viewer. Once authentication via Okta is completed, every new user is a Viewer. Only the Admin can assign a different role (Editor or Admin). The Viewer can search, read and bookmark documents.
  • Editor. As Editor, you can create drafts and update them as many times you would like. Only an Admin can publish a draft and make it available to the rest of the world. This is a common mechanism to preserve the quality of the articles and enforce a review process. Once an article is published and you want to improve, fix or correct it, you can do that as Editor. The article will be reverted to the state of draft and will need to be reviewed and published by an Admin. Editors can see their own drafts only. Editors can also tag documents.
  • Admin. The administrator can create, edit and publish all the documents; can assign roles and archive and delete documents. Only the Admin can create new document tags.

Technology

Python, Redis Stack, Toast UI Web Editor, Flask Web framework, JQuery, JQueryUI, Notify.js, Bulma CSS framework, Okta authentication

Feel free to download, use, change and reuse the source code. Feedback is welcome.
