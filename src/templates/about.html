What is keybase?

This software is a Knowledge Base written in Python. For this platform, I have used:

  • Redis Stack, using RediSearch for full text searches and vector similarity
  • Toast UI Web Editor, used as the Markdown editor
  • Flask Web framework
  • JQuery
  • JQueryUI
  • Notify.js plugin for JQuery
  • Bulma CSS framework
  • Okta authentication

Feel free to download, use, change and reuse the source code. Feedback is welcome.
